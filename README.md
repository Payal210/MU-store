Online Shopping Application

A full-stack e-commerce web application built using React.js for the frontend and Express.js for the backend. The platform provides a seamless online shopping experience where users can browse products, manage their cart, place orders, and track purchases.

##Features
User Authentication
User Registration
Secure Login & Logout
Session/JWT-based Authentication
Role-based Access (Admin/User)
Product Management
View all products
Product details page
Product search and filtering
Admin product management (Add, Update, Delete products)
Shopping Cart
Add products to cart
Update product quantities
Remove items from cart
View cart summary and total price
Order Management
Place orders
View order history
Track order status
Admin order management
User Profile
Manage account information
View previous orders
Update personal details

##Frontend
React.js
React Router
Axios
Context API / Redux (if used)
CSS / Bootstrap / Tailwind CSS

##Backend
Express.js
Node.js
MongoDB with Mongoose
JWT Authentication
Express Session
